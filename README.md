# Plataforma-Comercial-B2B-Direcionada
Projeto de site atacadista sem checkout tradicional e sem integração direta de estoque, estruturado com API para organização de pedidos e finalização via WhatsApp. Inclui app interno para acompanhamento de faturamento e controle operacional realizado por planilhas Excel/Sheets, formando um modelo híbrido de vitrine digital e conversão assistida.
